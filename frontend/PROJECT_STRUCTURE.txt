Component Hierarchy:
====================

Pages (Templates)
  └── CascadingSelect (Organism)
      └── SelectGroup (Molecule)
          └── Select (Atom)
              └── MUI Components

Custom Hook: useCascadingSelect
  - Manages selection state
  - Computes available options
  - Handles nested navigation

Design Patterns:
================

1. Atomic Design - Clear component hierarchy
2. Separation of Concerns - UI vs Logic
3. Mobile-First - Responsive breakpoints
4. Container/Presenter - Smart vs Dumb components
5. Custom Hooks - Reusable logic
6. Type Safety - Full TypeScript coverage
7. Accessibility First - ARIA, keyboard nav

Responsive Breakpoints:
=======================

- Mobile:  < 640px
- SM:      640px+
- MD:      768px+
- LG:      1024px+
- XL:      1280px+
