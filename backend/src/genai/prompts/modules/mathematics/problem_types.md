## üéØ MATCHING PROBLEMS TO CURRICULUM TOPICS

**CRITICAL: The problem type MUST match the curriculum topic specified.**

### Step 1: Identify the Core Mathematical Skill
Read `curriculum_topic.name` and `example_tasks` to determine:
- What type of math is this? (algebra, geometry, statistics, etc.)
- What specific operation/skill? (solving equations, calculating area, finding probability, etc.)
- What level of complexity? (basic, intermediate, advanced)

### Step 2: Choose Appropriate Problem Structure
- **Solving equations** ‚Üí Use algebra structure (unknowns, constraints, solve for variable)
- **Calculating area/volume** ‚Üí Use geometry structure (shapes, dimensions, measurements)
- **Finding probability** ‚Üí Use statistics structure (events, likelihood, data)
- **Optimization** ‚Üí Use calculus/function structure (maximum/minimum, rates)

### Step 3: Select Realistic Professional Context
Find a profession that ACTUALLY uses this type of math:
- Don't force geometry into a statistics context
- Don't force calculus into a basic arithmetic context
- Match the mathematical complexity to the professional scenario

## üö´ COMMON MISTAKES TO AVOID

‚ùå **Wrong Problem Type for Topic**
- Topic: Quadratic equations
- Mistake: Creating a geometry problem that doesn't need quadratic equations

‚ùå **Insufficient Data**
- Topic: Calculating volume
- Mistake: Only providing 2 dimensions of a 3D shape

‚ùå **Unrealistic Numbers**
- Topic: Projectile motion
- Mistake: Bullet traveling at 5000 m/s (unrealistic for handgun)

‚ùå **Circular Reasoning**
- Topic: Solving for unknown variable x
- Mistake: Providing x in the problem setup

‚ùå **Ambiguous Questions**
- Bad: "Estimate how long approximately..."
- Good: "Calculate the exact time in seconds when..."
