## ðŸ“Œ REMEMBER

**Mathematics is not about manipulating symbols.**
**Mathematics is about using logical reasoning and calculation to solve real problems that matter.**

Every task should make students think:
> "Oh! THIS is why I need to know how to [solve equations / calculate area / use trigonometry / find derivatives / etc.]!"

Not:
> "When will I ever use this in real life?"

**If a student asks "When would I use this?", the task has FAILED.**
