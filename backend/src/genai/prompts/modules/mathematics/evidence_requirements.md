## üìê DATA REQUIREMENTS BY PROBLEM TYPE

### **Geometry Problems**
Provide:
- All relevant dimensions (length, width, height, radius, etc.)
- All relevant angles (in degrees or radians as specified)
- Shape descriptions (right triangle, equilateral, rectangle, etc.)
- Scale factors if using diagrams

### **Algebra/Equation Problems**
Provide:
- All known variable values
- All coefficients
- All constraints or boundary conditions
- Context for what unknowns represent

### **Function/Graph Problems**
Provide:
- Function equation OR sufficient data points
- Domain restrictions if any
- Initial conditions or boundary values
- What the variables represent in context

### **Probability/Statistics Problems**
Provide:
- Sample size or population size
- Event probabilities or frequencies
- All relevant data values
- Conditions for events

### **Trigonometry Problems**
Provide:
- At least 3 pieces of information about triangles (sides/angles)
- Angle measurements in specified units
- Which angle/side needs to be found
- Context (navigation bearing, slope angle, etc.)

### **Calculus Problems**
Provide:
- Function equation describing the system
- Initial conditions
- Bounds of integration if needed
- What is being optimized/analyzed

### **Finance/Percentage Problems**
Provide:
- Principal/initial amount
- Interest rate or percentage change
- Time period
- Compounding frequency if applicable

## üéØ SPECIAL CASES

### **When Diagrams Are Referenced**

If you mention a diagram, chart, or visual:
- Describe it precisely in text
- Provide all measurements shown in the diagram
- State what the diagram illustrates
- Give dimensions, angles, or data points visible in it

**Example:**
"The site plan shows a triangular plot of land with measurements labeled: north side = 47.2 meters, east side = 53.8 meters, and the angle between them = 78 degrees."

### **When Multiple Solutions Are Possible**

If the problem has multiple valid solutions:
- State this explicitly
- Provide criteria for selecting among solutions
- Ask students to identify ALL solutions or the most appropriate one

**Example:**
"The quadratic equation will yield two time values. Identify both solutions and explain which one is physically meaningful for this scenario (time cannot be negative)."

### **When Real-World Variation Exists**

If there's inherent uncertainty in real scenarios:
- Provide specific values to use for the calculation
- Note the real-world variation in context, but give precise values for the math

**Example:**
"The steel's tensile strength typically ranges from 1,650-1,850 MPa depending on manufacturing. For this analysis, use the specification value of 1,770 MPa."
